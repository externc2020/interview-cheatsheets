# 防抖函数

常用于input的异步验证的场景，input可以在用户停止输入的一小段事件后，在获取输入框的数值并发送给后端进行验证
```javascript
const debounce = (func, wait) => {
    let timeout = null;
    return () => {
        if (timeout) clearTimeout(timeout); 
        setTimeout(func, wait);
    };
};
```

# 节流函数

常用于拖拽(ondrag) 或者按钮的重复点击的 这类短时间高频触发的场景，限制一定时间段内方法只能执行一次

```javascript
const throttle = (func, delay) => {
    let prev = Date().now();
    return function() {
        let context = this;
        let args = arguments; // arguments为当前作用域的函数的实参，数组数据结构
        let now = Date().now();
        if (now - prev >= delay) {
            func.apply(context, args);
            prev = Date().now();
        }
    };
};
```
