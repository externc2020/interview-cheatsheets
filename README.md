interview-cheatsheets
=====================

本项目旨在提供一个软件工程师面试复习清单。本着“少废话，纯干货“的原则，尽可能呈现各块知识要点，做到有广度的同时也不乏深度上的扩展。同时因为维护者的能力水平有限，错误和纰漏难以避免，望广大网友不吝指教，共同完善本仓库。
